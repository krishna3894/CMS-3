== Description ==
Custom Post Type(Game reviews).
In this plugin the user can type the comment and it can be posted on the website.
As basically the array is called and any message typed will give the display.
The post can be of any characters and the typed message will be displayed.
The icon is also added to it as to display it.
The main aim is to use it for creating a review.

The plugin can be directly installed by uploading the file to the wordpress. 
After the plugin is installed it needs to get activated so that it can be used.
After the plugin is activated the post needs to be given a name and post needs to be written and 
published so that the post can be seen in live feed.
